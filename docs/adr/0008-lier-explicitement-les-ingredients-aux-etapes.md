# Lier explicitement les ingrédients aux étapes

Les Étapes de préparation référencent explicitement les Ingrédients et peuvent préciser une quantité d’usage. Les interfaces publiques affichent ces informations dans des pastilles toujours visibles plutôt que de les cacher dans des popovers ou de déduire automatiquement les correspondances depuis le texte, car la saisie éditoriale validée évite les ambiguïtés et reste utilisable sur mobile pendant la cuisine.
