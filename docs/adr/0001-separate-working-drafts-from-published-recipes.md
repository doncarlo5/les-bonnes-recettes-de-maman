# Séparer les brouillons de travail des recettes publiées

Les autosauvegardes de l'admin sont stockées dans un Brouillon de travail distinct de la Version publiée. Cette séparation évite qu'une modification mobile incomplète change immédiatement le site public, limite les écritures fréquentes sur les documents lus par le site et permet une publication atomique après la revue éditoriale.
